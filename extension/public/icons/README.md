# Icons Directory

Place your extension icons here with these exact filenames:

- `icon16.png`  — 16×16 px  (used in browser toolbar)
- `icon48.png`  — 48×48 px  (used in extensions page)
- `icon128.png` — 128×128 px (used in Chrome Web Store)

## Quick Way to Generate Icons

You can use any image editor (e.g., Adobe Express, Canva, GIMP) to export your
logo at each size, or use an online favicon generator like:
- https://favicon.io  
- https://realfavicongenerator.net

Until you add icons the extension will load without an icon badge —
that's fine for development.
